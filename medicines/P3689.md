* When used as an irrigation solution, use an appropriate amount as directed by a physician. Apply an appropriate amount for cleaning skin, wound surfaces, and mucous membranes, wet dressings, gargling, and nebulization/inhalation. Used for cleaning medical devices.

### ⚠️ 크린클관류제(염화나트륨) - Precautions
* No information available
* Before using this medicine, patients with severe heart failure or kidney failure should consult a doctor or pharmacist. This medicine should be used topically and not as a parenteral injection. Do not use if systemic absorption may occur. Do not use if there are suspended particles or if the container is damaged. When used for irrigation, use a newly opened package and do not reuse any remaining solution.
