### 🕒 중외관류용멸균증류수 - Dosage and Administration

### ⚠️ 중외관류용멸균증류수 - Precautions
*   No information.
*   This medicine must be used topically and should not be used as a parenteral injection. This medicine is a hypotonic solution and may cause hemolysis, so its use in surgical procedures is not recommended. Patients with severe heart failure or kidney failure should consult a doctor or pharmacist before using this medicine. Do not use if particulate matter is present or the container is damaged. When used for irrigation, use a freshly opened solution and do not reuse any remaining solution.
