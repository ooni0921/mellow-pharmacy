*   When used as an irrigation solution, use an appropriate amount as directed by a physician.

Use for cleaning medical devices.

### ⚠️ 세비안관류제(염화나트륨) - Precautions
*   No information
*   Before using this medicine, patients with severe heart failure or kidney failure should consult a doctor or pharmacist.

This medicine should be used topically and not as a parenteral injection.

Do not use if systemic absorption occurs.

Do not use if there are suspended particles or if the container is damaged.

When used for irrigation, use a freshly opened product and do not reuse any remaining solution.
